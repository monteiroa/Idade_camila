# OBI 2021 – Idade de Camila

## Descrição da questão
Dado três idades diferentes, descobrir qual é a idade intermediária (nem a maior, nem a menor).


## Estratégia utilizada
O programa lê as três idades e utiliza estruturas condicionais (if/elif/else)
para verificar qual das três idades está entre as outras duas.

A lógica verifica se cada idade é maior ou igual a uma e menor ou igual à outra.
A idade que satisfaz essa condição é a idade intermediária de Camila.
